# Regex Data Validation in Python

This project demonstrates how to use **Regular Expressions (Regex)** in Python to validate common user inputs such as **email addresses, Indian mobile numbers, and passwords**. The program uses the built-in `re` module and follows clean coding practices with reusable validation functions.

##  Features
- Email validation using industry-standard regex
- Indian mobile number validation (10 digits, starts with 6–9)
- Strong password validation with multiple rules
- Dynamic user input handling
- Clear success and error messages
- Edge case handling (empty input, invalid formats)

##  Tools & Technologies
- Python 3.13.9
- `re` module (built-in)
- VS Code / Jupyter / PyCharm Community

##  Validation Rules
### Email
- Proper username and domain format
- Example: `user@example.com`

### Mobile Number
- Exactly 10 digits
- Starts with digits 6–9

### Password
- Minimum 8 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one digit
- At least one special character (@$!%*?&)
